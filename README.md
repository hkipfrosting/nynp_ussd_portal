# School USSD Portal

A USSD-based school portal that allows students to access school services via mobile phone.

## 📱 USSD Code
Dial **\*254#** to access the portal

## ✨ Features
- Check fees balance
- View exam results
- Attendance reports
- School calendar
- Contact teacher
- Report absence
